Model Context Protocol (MCP) is an open standard from Anthropic that lets AI assistants connect to data sources, tools and services through a consistent interface.

## What assess means here

The AI Capability and Enablement team is evaluating MCP. It is not yet ready for use in Defra delivery without prior consultation.

## Why we are watching it

MCP could simplify how Defra services expose data and tools to AI agents, reducing the bespoke integration work each team would otherwise face.

## Before you start

Do not use MCP in delivery without talking to the AI Capability and Enablement team. They can share what the evaluation has found so far and help you avoid common pitfalls.

## Learn more

Read the [Model Context Protocol overview](https://modelcontextprotocol.io/overview) on the official site.
