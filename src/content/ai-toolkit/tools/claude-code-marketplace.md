The Claude Code plugin marketplace lets teams install third-party extensions that add tools, skills and integrations to Anthropic's coding agent.

## What assess means here

The AI Capability and Enablement team is evaluating the marketplace. Plugins are not yet approved for use in Defra delivery without prior consultation.

## Why we are watching it

Plugins can speed teams up considerably, but they also expand the trust boundary of the agent. The team is working out which plugins are safe and how to vet them.

## Before you start

Do not install plugins for delivery work without talking to the AI Capability and Enablement team. They can share which plugins have been vetted so far.
