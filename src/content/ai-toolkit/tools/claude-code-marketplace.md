The Claude Code plugin marketplace lets teams install third-party extensions that add tools, skills and integrations to Anthropic's coding agent.

## What data you can use with it

<div class="govuk-inset-text">
  <p class="govuk-body govuk-!-margin-bottom-0">What a plugin can do with your data depends on the plugin and your data's classification. Check <a class="govuk-link" href="/ai-toolkit/guidance/using-data-with-ai">Using data with AI</a> before you use it.</p>
</div>

## Why we are exploring it

The AI Capability and Enablement team is evaluating the marketplace. Plugins are not yet approved for delivery.

Plugins can speed teams up considerably, but they also expand the trust boundary of the agent. The team is working out which plugins are safe and how to vet them.

Talk to the team before installing plugins for delivery work. They can share which have been vetted so far.
