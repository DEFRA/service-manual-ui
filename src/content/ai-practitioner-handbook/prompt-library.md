The Defra AI Prompt Library is a place for practitioners to share prompts, workflows and agent configurations. Anyone working on AI at Defra can browse and contribute.

Each entry includes the thinking behind it. You can see what the creator was trying to achieve, why the prompt works, and what you might learn from it.

## Featured entries

- **IPAFFS Utils** — a repository of SDLC skills for working with the IPAFFS codebase
- **AI Prototyping Workflow** — a 3-agent pipeline for building GOV.UK prototypes
- **Backlog Refinement from Transcript** — turns meeting recordings into structured backlog items
- **WebDriverIO test rules** — agent configuration for generating browser automation tests

## Defra and GDS aligned agent rules

Dan Roszkowski from Capgemini is working on official agent rules aligned with Defra, GDS and CDP standards. These will give teams a consistent starting point for configuring AI agents.
